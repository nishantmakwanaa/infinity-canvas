# CNVS PWA/Desktop Port

This folder documents the desktop-installable web app configuration for `https://canvas.nishantmakwana.tech`.

Runtime files used by the live app:

- `public/manifest.webmanifest`
- `public/sw.js`
- `public/pwa-192.png`
- `public/pwa-512.png`

When installability criteria are met, Chromium-based browsers surface the **Install app** option near the address bar.
